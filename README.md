# Homepage Configuration

This repository contains the configuration files for my **[GetHomepage](https://gethomepage.dev/)** dashboard.

## Configuration Files

| File | Purpose |
|------|---------|
| `services.yaml` | Defines the apps/containers displayed on the dashboard. |
| `widgets.yaml` | Configuration for information widgets. |
| `bookmarks.yaml` | Static links to external sites or unmanaged services. |
| `docker.yaml` | Connection settings for the Docker Socket. |
| `settings.yaml` | Global settings. |
| `custom.css` | Custom CSS overrides for theming. |
| `custom.js` | Custom JavaScript injections. |
